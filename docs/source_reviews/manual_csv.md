# Manual CSV

- Source name: Manual CSV Import
- Source URL: Local file provided by analyst or approved export source
- Access method: Local file drop
- Terms reviewed? yes
- Automated access allowed? yes
- Rate limits: Not applicable for local files
- Data fields expected: record_id, name, address, phone, email
- Privacy concerns: User-supplied files may contain personal or restricted data; analysts must exclude private or unauthorized records
- Recommended use: Prefer manually exported public/open datasets and keep all imports local to the workspace
- Notes: This connector does not fetch external data. Review the origin of each manual file before import.
